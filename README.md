Brand:I 앱 홍보를 위한 정적 웹페이지입니다.

## 구조

```
marketing-brandi/
├── index.html          # 메인 랜딩 페이지
├── images/             # 이미지 폴더
│   └── brandi-app-main.png  # 앱 목업 이미지
└── README.md
```

## 로컬 실행
